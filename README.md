# hello_agents

## Getting Started

### Prerequisites

- Python 3.x(Experimented with 3.13.9)

### Setup Virtual Environment

1. Create a virtual environment:
    ```bash
    python3 -m venv .venv
    ```

2. Activate the virtual environment:

    ```bash
    source .venv/bin/activate
    ```

### Running the Agent

Run the agent script:

```bash
python xxx.py
```
